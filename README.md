# BJTU_MAIL_SHOOTER

一个用于北京交通大学（BJTU）学生半自动投递推免/套磁邮件的 Python 脚本。支持动态称呼、自动挂载附件及无限轮询输入。

欢迎fork

## 功能特点

- **动态称呼**：每次发送前可手动输入老师姓名，正文自动替换。
- **自动附件**：脚本会自动扫描当前目录下的所有文件（除脚本本身外），将其作为附件发送。
- **SSL 安全连接**：默认使用 BJTU 邮箱的 SMTP SSL 端口（465）。
- **交互式操作**：通过无限循环实现批量投递，无需重复运行脚本。

## 使用方法

### 1. 克隆仓库
```bash
git clone https://github.com/TirpitzA/BJTU_MAIL_SHOOTER.git
cd BJTU_MAIL_SHOOTER
```

### 2. 配置脚本
打开 `send.py`，根据代码中的注释修改以下内容：
- `sender_email`: 填写你的北交邮箱（例如 `xxxxxx@bjtu.edu.cn`）。
- `subject_template`: 修改邮件主题模板。
- `body`: 修改邮件正文内容，将其中的 `***` 替换为你的真实个人信息。

### 3. 准备附件
将你需要随信发送的材料（如个人简历、成绩单、论文等）**放置在与 `send.py` 同一目录下**。
> **注意**：脚本会将该目录下除 `send.py` 以外的所有文件都作为附件发送，请确保目录中没有无关文件。

### 4. 运行脚本
```bash
python send.py
```
运行后，按照提示：
1. 输入第三方邮箱登录密钥（如果嫌麻烦可以把密钥直接改为硬编码，反正是自己用）
2. 输入收件人老师的邮箱。
3. 输入老师的姓名。

-北交邮箱密钥获取方式：在邮箱界面点击头像-->设置界面-->安全设置-->客户端专用密码

### 5. 退出
在提示输入邮箱时，输入 `q` 即可退出。

## 注意事项

- **安全性**：脚本中已隐去个人敏感信息，除了自己用建议不要在代码中硬编码密码。
- **SMTP 限制**：频繁发送可能会触发布件箱的反垃圾邮件策略，请合理安排发送间隔。
- **环境要求**：Python 3.x，无需额外安装第三方库（仅使用标准库）。
